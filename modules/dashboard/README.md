# Dashboard Module

## Structure
```
modules/dashboard/
├── components/
│   ├── DashboardStats.tsx
│   ├── RecentActivity.tsx
│   ├── LearningProgress.tsx
│   └── QuickActions.tsx
├── pages/
│   └── index.tsx
├── hooks/
│   └── useDashboard.ts
├── services/
│   └── dashboardService.ts
├── types/
│   └── dashboard.types.ts
└── index.ts
```

## Components
- **DashboardStats.tsx**: Dashboard statistics display
- **RecentActivity.tsx**: Recent activity feed
- **LearningProgress.tsx**: Learning progress overview
- **QuickActions.tsx**: Quick action buttons

## Pages
- **index.tsx**: Main dashboard page

## Hooks
- **useDashboard.ts**: Hook for dashboard data

## Services
- **dashboardService.ts**: Dashboard business logic

## Types
- **dashboard.types.ts**: Dashboard type definitions
